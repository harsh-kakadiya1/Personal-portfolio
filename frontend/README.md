# Personal Portfolio

My personal portfolio website — modern, interactive, built with React, Framer Motion, and Tailwind. Frontend-only; no backend.

## Tech

- React, Vite, Tailwind CSS, Framer Motion
- Pages: About, Projects, Research, Playground, Contact

## Structure

```
src/
├── components/   # UI components
├── contexts/     # React context (e.g. project modal)
├── hooks/        # Custom hooks
├── pages/        # About, Projects, Research, Playground, Contact
├── services/     # API / services
└── lib/          # Utilities
```
